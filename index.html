Raintale allows anyone to create stories from archived web pages (mementos).

Get Started with Raintale

Raintale's Core Features

Raintale was developed with customization in mind.

File Output

Post your story to files in HTML, Jekyll, Markdown, and MediaWiki formats. With templates, you can create any single page story format you wish. An experimental video storyteller is also available.

Social Media Publishing

Use the Twitter storyteller to post your as a Twitter thread. An experimental Facebook storyteller is also available.

Presets and Templates

You can control the look of your story. Visit the Template Gallery to see examples.
